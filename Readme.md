#The MyWallet App

##Services
Initial thoughts are that these are the services inlcuded in the app:

###myAccount (port: 8080)
1. read users account
2. update users account
3. create new users account
4. delete users account

###myUser (port: 8081)
1. create new user
2. update user
3. read users info
4. delete user
5. userRoles
-- Customer
-- Admin

###myLogin (port: 8082)
1. login service
2. logout service
3. Validation
4. security

###myHistory (port: 8083)
1. read history
2. update history
3. create new history
4. delete history
